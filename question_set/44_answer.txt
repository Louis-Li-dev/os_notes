1. buddy system:

2. slab allocation
- 一個或更多 contiguous 的 frames 組成 slabs
- 一個 cache 由多個 slabs 組成並且一個 cache 支援一個 kernel data structure
- 優點
    - no fragmentation
    - fast memory request satisfaction 因為 object 事先建立好，用完 free 但 cache 還在裡面
